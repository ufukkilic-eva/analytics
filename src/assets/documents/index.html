14. Soft Color System

Definition:
Soft tokens represent low-emphasis background layers. They are NOT used for primary text colors.
Token Value Source Usage
--brand-blue-soft alias of --brand-blue-muted Soft badges, active pills, soft highlights
--brand-purple-soft alias of --brand-purple-muted Dropdown active background, soft selections
Backward Compatibility:
--brand-blue-muted and --brand-purple-muted are legacy aliases kept for backward compatibility. New implementations MUST use soft naming.
